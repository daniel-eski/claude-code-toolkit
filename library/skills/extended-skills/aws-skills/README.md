Building AI agents that leverage AWS services like Bedrock, Lambda, and Step Functions.

### aws-cdk-development
Infrastructure automation using TypeScript/Python with AWS CDK.

### aws-cost-operations
Cost analysis, budgeting, and optimization across AWS accounts.

### aws-serverless-eda
Event-driven systems using EventBridge, SQS, SNS, and Lambda.
