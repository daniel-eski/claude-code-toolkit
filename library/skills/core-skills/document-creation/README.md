# Document Creation Skills

Skills for creating and manipulating documents from [anthropics/skills](https://github.com/anthropics/skills).

## Skills

| Skill | Format | Description |
|-------|--------|-------------|
| docx | Word | Create, edit, and analyze Word documents |
| doc-coauthoring | Word | Collaborative document editing |
| pptx | PowerPoint | Create, edit, and analyze presentations |
| xlsx | Excel | Create, edit, and analyze spreadsheets |
| pdf | PDF | Extract text, create PDFs, handle forms |

## Usage Patterns

### Creating Documents

```
Create a Word document outlining our Q1 product roadmap
```

### Analyzing Documents

```
Read the attached Excel file and summarize the sales data
```

### Converting Formats

```
Convert this markdown to a formatted PDF with our brand colors
```

## Dependencies

These skills may require additional libraries:
- `python-docx` for Word documents
- `openpyxl` for Excel spreadsheets
- `python-pptx` for PowerPoint
- `PyPDF2` or similar for PDF handling

Check individual SKILL.md files for specific requirements.
