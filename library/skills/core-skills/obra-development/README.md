# Obra Development Skills

Software development workflow skills by [Jesse Vincent](https://github.com/obra).

## Source

[obra/superpowers](https://github.com/obra/superpowers)

## Skills

| Skill | Description |
|-------|-------------|
| test-driven-development | Write tests before implementing code |
| subagent-driven-development | Development using multiple coordinated sub-agents |
| systematic-debugging | Methodical problem-solving in code |
| finishing-a-development-branch | Complete Git branches for merge |
| requesting-code-review | Initiate code review processes |
| receiving-code-review | Process and incorporate code feedback |
| using-git-worktrees | Manage multiple Git working trees |
| verification-before-completion | Validate work before finalizing |
| writing-skills | Develop and document new Claude Code skills |

## Workflow Integration

These skills form a complete development workflow:

1. **Planning**: Use `writing-plans` from obra-workflow
2. **Development**: `test-driven-development` + `subagent-driven-development`
3. **Debugging**: `systematic-debugging`
4. **Git workflow**: `using-git-worktrees` → `finishing-a-development-branch`
5. **Review**: `requesting-code-review` → `receiving-code-review`
6. **Finalization**: `verification-before-completion`
