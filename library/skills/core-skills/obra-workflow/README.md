# Obra Workflow Skills

Strategic planning and workflow methodology skills by [Jesse Vincent](https://github.com/obra).

## Source

- Main repository: [obra/superpowers](https://github.com/obra/superpowers)
- Experimental skills: [obra/superpowers-lab](https://github.com/obra/superpowers-lab)

## Skills

| Skill | Description |
|-------|-------------|
| brainstorming | Generate and explore ideas through structured thinking |
| writing-plans | Create strategic documentation and project plans |
| executing-plans | Implement and run strategic plans step by step |
| dispatching-parallel-agents | Coordinate multiple simultaneous sub-agents |
| using-superpowers | Leverage core platform capabilities effectively |
| using-tmux-for-interactive-commands | Control interactive CLI tools (vim, git) via tmux |

## Philosophy

These skills follow Jesse Vincent's "superpowers" methodology for AI-assisted development:

1. **Structured thinking**: Break complex tasks into manageable steps
2. **Parallel execution**: Use sub-agents for concurrent work
3. **Verification**: Validate outputs before proceeding
4. **Interactive control**: Handle CLI tools that require human interaction
