## Planning Steps

### 1. Understanding
- Restate the goal in your own words
- Define what success looks like

### 2. Context Assessment
- What context is available (files, conversation)?
- What needs discovery?
- Ambiguities requiring clarification?

### 3. Approach Options
Present 2-3 approaches with:
- Description
- Tradeoffs (complexity, risk)
- Scope (files/systems involved)

Recommend one with rationale.

### 4. Risks
- Potential issues?
- Assumptions to validate?
- Prerequisites to check?

### 5. Alignment
Confirm before proceeding:
1. Is understanding correct?
2. Approach approved?
3. Constraints or preferences?

**Wait for user confirmation before implementation.**
