# Progress

> Status tracking and session logs.

## What's Here

| File/Folder | Purpose | How to Use |
|-------------|---------|------------|
| `current-status.md` | Current state, next actions | Read first; update each session |
| `session-logs/` | Per-session work notes | Add dated file each session |

## current-status.md

**Read this first** when continuing work.

Contains:
- What phase we're in
- What's done
- What's in progress
- What's blocked
- Next actions

**Update before ending your session.**

## Session Logs

Create one file per session: `session-logs/YYYY-MM-DD-description.md`

Include:
- What you worked on
- What you completed
- Issues encountered
- Handoff notes
